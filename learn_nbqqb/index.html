教你NeoBasicQQBot


提示

下文中的NBQQB代指NeoBasicQQBot

要求

你必须拥有Python 3.11或更高版本才能使用NBQQB

安装

安装NBQQB非常简单，你只需要在命令行输入以下命令：

pip install nbqqb

使用

使用NBQQB非常简单，你只需要在Python文件中输入这些代码并修改：

from nbqqb import NBQQBot

def main() -> int:
    keyword: str = "改为你聊天窗口标题包含的关键词"
    bot: NBQQBot = NBQQBot(keyword)

    if not bot.find_window():
        print("啊哦，没有找到窗口！")
        return 1

    # 发送一条消息
    bot.send_message("嘻嘻")

    # 可选：如果保持窗口不切换，可以继续用 quick_send 快速发送多条
    # bot.quick_send("这是第二条消息")
    return 0


if __name__ == "__main__":
    main()

尝试运行你的Python文件，看看是否能成功发送消息！

注意

NBQQB是一个低级QQ机器人，实现方法是模拟人类操作QQ，性能较差，功能较少，但风控风险低
